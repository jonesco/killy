Colors

CSS variables and key hex values. Use var(--name) in styles.

--ink: #0b1220 — primary text
--muted: #5a6573 — secondary text, placeholders
--border: #d6dde6 — borders
--brand: #1f4acc — links, accents
--blue: #e6f0ff — light blue tint
--accent: #ea580c — primary button, links, success
--accent-hover: #c2410c — primary hover
--accent-light: #fb923c — sidebar active
#13183f — sidebar background
#fff — cards, inputs, modal
#f2f2f2 — page background (with bg image in app)

Typography

Trade Gothic (300, 400, 700). Body and inputs use the same font stack; inputs use a monospace stack.

font-family: "Trade Gothic", system-ui, -apple-system, "Segoe UI", Roboto, Helvetica, Arial, sans-serif;

Weights:

Light (300) — Trade Gothic Light

Regular (400) — Trade Gothic

Bold (700) — Trade Gothic Bold

Sizes:

h1 — 30px, letter-spacing -0.02em

h2 — 23px

Body — 15px default

Small / muted — 15px, color var(--muted)

Inputs: SF Mono, Monaco, Inconsolata, Roboto Mono, Source Code Pro, Menlo, Consolas, DejaVu Sans Mono

Responsive

Breakpoint 768px. Notable overrides: grid becomes single column; preview column hidden on mobile; container padding 24px 12px; card padding 12px; h1 25px; actions wrap, primary button full width.

@media (max-width: 768px) { .grid { grid-template-columns: 1fr; } .grid > div:last-child { display: none; } }
